gitインストール

  1. git for windowsをダウンロード
  2. ダウンロードしたインストールファイルを実行
  3. インストール完了後、ユーザー名・メールアドレスをgitに登録 
              > git config --global user.name "USER NAME"
          > git config --global user.email "EMAIL"
